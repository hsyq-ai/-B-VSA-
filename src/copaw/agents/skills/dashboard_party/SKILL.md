# 党建学习（SOP）

## 目标
根据用户时间与重点方向，输出学习计划、要点与自测题。

## 适用场景
- 每周学习安排
- 主题学习汇报

## 输入要求
1. 学习周期（本周/本月）
2. 重点方向（如政策、理论、案例）
3. 可用时间

## 工作流程
1. 确认周期与重点
2. 输出学习计划（分天/分阶段）
3. 提供要点摘要
4. 给出自测问题

## 输出模板
【学习计划】
- 第 1 阶段：
- 第 2 阶段：

【学习要点】
1.
2.
3.

【自测问题】
1.
2.

## 质量检查
- 计划是否可执行
- 要点是否清晰
- 自测是否覆盖重点
