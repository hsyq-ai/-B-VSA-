# 当前任务梳理（SOP）

## 目标
基于用户记忆档案与近期记录，形成可执行的任务盘点与优先级列表。

## 适用场景
- 每日/每周任务盘点
- 需要澄清当前工作重心
- 需要识别阻塞与风险

## 输入要求
- 默认使用用户记忆档案
- 如记忆不足，可询问：
  - 本周重点目标
  - 近期最紧急事项

## 工作流程
1. 从记忆档案中提取任务线索
2. 归类为进行中/待确认/阻塞
3. 按优先级排序（高/中/低）
4. 为每项补充下一步与负责人
5. 给出本日/本周执行建议

## 输出模板
【高优先级】
1. 任务：
   - 下一步：
   - 阻塞：

【中优先级】
1. 任务：
   - 下一步：

【低优先级】
1. 任务：

【建议】
- 今日建议：
- 本周建议：

## 质量检查
- 是否覆盖主要任务
- 优先级是否合理
- 每项是否有下一步
- 是否识别关键阻塞
