# 已办结总结（SOP）

## 目标
基于用户记忆档案，输出已完成事项清单与关键贡献，便于复盘与汇报。

## 适用场景
- 周报/月报
- 绩效复盘
- 领导汇报

## 输入要求
- 默认使用用户记忆档案
- 如需范围限定，询问时间范围

## 工作流程
1. 识别已完成事项
2. 按时间或项目归类
3. 提炼结果与影响
4. 汇总可复用经验

## 输出模板
【已办结清单】
1. 事项：
   - 完成时间：
   - 结果：

【关键成果】
- 成果 1：
- 成果 2：

【可复用经验】
- 经验 1：
- 经验 2：

## 质量检查
- 是否聚焦“已完成”
- 是否描述结果与价值
- 是否结构清晰、便于汇报
