# 公文写作（SOP）

## 目标
快速生成规范、公文格式清晰的草稿，并给出可替换措辞。

## 适用场景
- 通知
- 请示
- 报告
- 总结

## 输入要求
按优先级确认：
1. 公文类型
2. 目的与对象
3. 核心要点（3-5 条）
4. 时间/地点/责任人（如适用）

## 工作流程
1. 收集关键信息
2. 选择结构模板
3. 生成草稿
4. 提供精简版与可选措辞

## 输出模板
【关键信息】
- 类型：
- 目的：
- 对象：
- 要点：

【公文草稿】
标题：
正文：
落款：

【可选措辞】
- 版本 A：
- 版本 B：

## 质量检查
- 结构是否符合公文类型
- 语气是否正式
- 关键信息是否完整
