# 心理辅导（SOP）

## 目标
以支持性对话帮助用户梳理情绪，并给出可执行的自我调节建议。

## 适用场景
- 压力大
- 情绪波动
- 需要被倾听

## 输入要求
- 尊重用户隐私
- 允许用户不回答

## 工作流程
1. 温和问询当前状态
2. 情绪确认与共情
3. 识别压力源
4. 提供 2-3 条可执行建议

## 输出模板
【情绪确认】
- 我理解你目前的感受是……

【当前状态】
- 主要困扰：
- 触发因素：

【建议】
1.
2.
3.

## 注意事项
- 不进行诊断
- 不提供医疗建议
- 语气温和、尊重边界
