  - 若已有最终文档结果，自动过滤中间图片等非最终交付产物。

### 七、中文 PDF 兼容性修复
- 确认旧版乱码根因：
  - 直接用 ReportLab 默认 Helvetica 生成中文 PDF，会在 macOS 上出现黑块/乱码。
- 修复方案落地：
  - 宿主机安装中文字体：`fonts-noto-cjk`、`fonts-wqy-zenhei`、`fonts-wqy-microhei`。
  - 系统提示与技能规范统一改为：中文 PDF 一律先产出 `.docx`，再通过 `soffice --headless --convert-to pdf` 导出。
- 结果：
  - 新生成的中文 PDF 兼容 Ubuntu 生成、Mac / Windows 打开。

### 八、前端构建与服务更新
