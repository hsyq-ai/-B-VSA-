# 3.21 升级方案（实施任务清单 + 代码落地步骤）

本文档用于指导 3.21 升级实施，涵盖提示词模板管理、导航结构升级、会话管理、文件下载与 ASR 交互优化等全部需求。内容按“执行任务清单 + 代码落地步骤”组织。

---

## 一、目标范围（本次升级包含）

1. 管理后台新增“提示词模板”管理入口与功能。
2. ASR 图标录音态反馈（录音中浅蓝色，结束恢复默认）。
3. 会话管理新增“删除/批量删除”。
4. 生成文件（Word/PDF 等）可在会话窗内下载。
5. 工作区新增“定时任务”入口。
6. 左侧“部门看板”更名为“企业看板”，并拆分红智秘书/组织架构层级。
7. 左侧新增“联络站”入口（发起/选定/部门/员工多级触发）。

## 三、代码落地步骤（模块级执行指引）

> 以下步骤为推荐落地顺序，可逐项提交。

### Step 1：提示词模板管理（后台）
**目标**：管理员可新增/编辑/禁用各按钮对应模板。

**后端（建议路径）**
- 新建模型/存储（建议 JSON 或 SQLite）
  - `template_id`
  - `display_name`
  - `trigger_key`
  - `prompt_text`
  - `scope`（admin/employee/all）
  - `is_active`
- 新增 API
  - `GET /api/prompt-templates`（列表）
  - `POST /api/prompt-templates`（新增）
  - `PUT /api/prompt-templates/{id}`（更新）
  - `DELETE /api/prompt-templates/{id}`（禁用/删除）

**前端（管理后台）**
- 新增导航入口：提示词模板
- CRUD 页面（表格 + 编辑弹窗）

---

### Step 2：导航栏触发绑定模板
**目标**：所有左侧按钮触发时，从模板加载 prompt_text。

**后端**
- 提供 `GET /api/prompt-templates/resolve?trigger_key=xxx`

**前端（员工侧）**
- 每个按钮绑定 `trigger_key`
- 点击后：
  1. 请求模板接口
  2. 新建会话
  3. 自动写入并提交 prompt_text

---

### Step 3：导航结构升级
**3.1 企业看板**
- 左侧 `部门看板` → `企业看板`
- 结构：
  - 红智秘书
    - 工作汇报（trigger_key: `corp_report`）
    - 任务下达（trigger_key: `corp_assign`）
  - 组织架构
    - 部门列表（管理员除外）
      - 部门状态（trigger_key: `dept_status_{deptId}`）
      - 员工列表（trigger_key: `dept_staff_{staffId}`）

**3.2 联络站**
- 新入口 `联络站`
- 结构：
  - 发起
    - 协同任务（trigger_key: `contact_collab`）
    - 活动事项（trigger_key: `contact_event`）
    - 通知会议（trigger_key: `contact_meeting`）
    - 投票（trigger_key: `contact_vote`）
  - 选定
    - 部门列表（管理员除外）
      - 员工列表（trigger_key: `contact_staff_{staffId}`）

**3.3 工作区新增定时任务**
- 新条目 `定时任务`（trigger_key: `schedule_task`）

---

### Step 4：会话管理删除
**后端**
- `DELETE /api/chats/{session_id}`
- `DELETE /api/chats?scope=all`

**前端**
- 会话列表新增“删除”按钮
- 顶部新增“批量删除全部”按钮（需确认）

---

### Step 5：文件生成与下载
**后端**
- 文档生成后落地到 `chat_files` 存储
- 返回 `file_id + download_url`

**前端**
- 会话消息中检测 `files` 字段
- 渲染“下载按钮”或“文件卡片”

---

### Step 6：ASR 图标录音态
**前端**
- `VoiceInputButton` 增加 `isRecording` 状态
- 录音时 icon class 切换为浅蓝
- 结束后恢复默认
