## 5. 关键环境变量

### 5.1 必须设置

```bash
# 工作目录（存放 DB、记忆、技能等运行时数据）
export COPAW_WORKING_DIR="/path/to/copaw_work"

# Python 模块搜索路径
export PYTHONPATH="/path/to/CoPaw/src"
```

### 5.2 TTS 配置（由 scripts/voice_secretary_tts_env.sh 加载）

```bash
export COPAW_TTS_ENABLED=1
export COPAW_TTS_PROVIDER=voxcpm2          # 或 edge-tts
export COPAW_TTS_VOXCPM_MODEL_DIR=/path/to/VoxCPM2
export COPAW_TTS_VOXCPM_DEVICE=cuda        # 或 cpu
```

### 5.3 可选

```bash
export COPAW_LOGIN_OUTBOX_DELIVERY=1       # 登录时投递收件箱消息
export COPAW_AUTH_SALT=your_salt           # 密码哈希盐值（默认: copaw_auth_salt）
```

## 7. 全新安装步骤

### 7.1 系统依赖

```bash
# Ubuntu
sudo apt-get update
sudo apt-get install -y python3.11 python3-pip nodejs npm sqlite3 lsof

# 或使用 Miniconda
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
bash Miniconda3-latest-Linux-x86_64.sh -b -p /environment/miniconda3
export PATH="/environment/miniconda3/bin:$PATH"
```

### 7.2 获取代码

```bash
git clone https://github.com/hsyq-ai/-B-VSA-.git /path/to/CoPaw
cd /path/to/CoPaw
```

### 7.3 后端安装

```bash
# 安装 Python 依赖
pip install -e .

# 安装 Playwright 浏览器（可选）
playwright install chromium

# 生成自签名 SSL 证书（如果没有）
mkdir -p certs
openssl req -x509 -newkey rsa:2048 -keyout certs/key.pem -out certs/cert.pem \
  -days 365 -nodes -subj "/CN=localhost"
```

### 7.4 前端构建

```bash
cd console
npm install
npm run build
cd ..
```

### 7.5 初始化运行时目录

```bash
export COPAW_WORKING_DIR="/path/to/copaw_work"
mkdir -p "$COPAW_WORKING_DIR"

# 首次启动会自动创建 auth.db、agent_os.db 等
# 第一个注册的用户自动成为管理员
```

### 7.6 配置 LLM API

编辑 `$COPAW_WORKING_DIR/.env`：

```env
# 示例：使用 DashScope (通义千问)
DASHSCOPE_API_KEY=sk-xxxxx

# 或 OpenAI
OPENAI_API_KEY=sk-xxxxx
OPENAI_BASE_URL=https://api.openai.com/v1
```

### 7.7 启动服务

```bash
# 一键构建+重启
bash CoPaw/4_16_build_restart.sh

# 或仅重启（跳过构建）
bash CoPaw/4_16_build_restart.sh --skip-build
```

### 7.8 访问

- 本地: `https://localhost:8088`
- 首次访问需注册账户（第一个自动成为管理员）

## 10. 常见问题

### Q: 登录 401 Unauthorized
**A:** 检查 `COPAW_WORKING_DIR` 环境变量是否正确设置。如果未设置，copaw 会使用 `~/.copaw/` 而非指定的数据目录，导致读取了不同的用户数据库。

### Q: 前端页面空白
**A:** 确认 `console/dist/` 目录存在（需要先 `npm run build`）。

### Q: IAP 转发任务失败
**A:** 检查 `agent_os.py` 日志，常见原因包括 LLM API Key 未配置、模型不可达。

### Q: TTS 无声音
**A:** 检查 VoxCPM2 模型路径和 GPU 可用性，或切换到 `edge-tts`（无需 GPU）。
